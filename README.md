# Android-fmlPlayer

    160421：
        Fix - 1、修改项目为gradle
              2、将BASS库打包为jar引用

基于BASS Library的封装，可直接用于Android

包含部分BASS FX的音效等

使用时先将AudioLibrary中的res/string的内容放到需要用到的工程的res下

BASS Library:http://www.un4seen.com/

功能：

    1、可对Ogg音频无缝循环（前提是你音频剪切得好）
    
    2、支持多种格式音频（需要添加插件，我完全没加插件:D）
    
    3、想到什么再写吧反正就是练手用的
